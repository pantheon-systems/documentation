1. From the Organization's Dashboard, select the site from the Sites list.
  2. Click the site's name to enter the Site Dashboard, then click the Billing tab.
  3. Click View All Plans to select a plan size and annual or monthly billing
  4. From the Enter Billing Information page, use the Send a Request button.
  5. Enter the recipient's email address and click the Send Request button. A link is immediately sent to the email address to pay through the secure site.
Note that when the user clicks the link to pay, they will be prompted to log in or create an account. Once an account is created, the user will need to log in again with the newly created user ID.
