PHP versions [8.3.30](https://www.php.net/ChangeLog-8.php#8.3.30) and [8.4.17](https://www.php.net/ChangeLog-8.php#8.4.17) are now available on the platform. These updates bring the latest bug fixes and security patches for your sites. The latest versions have already rolled out to all eligible sites.

PHP 8.3.30 is available on both PHP Runtime Generation 1 and [PHP Runtime Generation 2](/php-runtime-generation-2).

PHP 8.4 is only available with the new [PHP Runtime Generation 2](/php-runtime-generation-2).
