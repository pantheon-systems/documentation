Terminus [4.1.5](https://github.com/pantheon-systems/terminus/releases/tag/4.1.5) is now available. This release adds new Elasticsearch search commands and deprecates the legacy Solr commands.

## Key improvements in this release

- **Elasticsearch search commands**: New `search:enable` and `search:disable` commands for managing search indexing ([#2783](https://github.com/pantheon-systems/terminus/pull/2783), [#2784](https://github.com/pantheon-systems/terminus/pull/2784))
  - Coming soon. These commands are currently non-functional until platform compatibility is added
- **Solr command deprecation**: `solr:enable` and `solr:disable` are now deprecated in favor of the new `search:enable` and `search:disable` commands
- **Bug fix**: Fixed empty body handling to use object instead of array ([#2780](https://github.com/pantheon-systems/terminus/pull/2780))

## How to upgrade to Terminus 4.1.5

If you use Homebrew (macOS-only) to manage your Terminus installation, you should upgrade using:

```shell{promptUser: user}
brew upgrade terminus
```

If you installed Terminus directly from the `.phar` file, you should upgrade using the `self:update` command:

```shell{promptUser: user}
terminus self:update
```

For more information about this release, visit the [GitHub release page](https://github.com/pantheon-systems/terminus/releases/tag/4.1.5).

If you have questions or concerns around Terminus, please use the [Terminus issue queue](https://github.com/pantheon-systems/terminus).
