---
title: "Drupal CMS 2 now available"
published_date: "2026-02-27"
categories: [drupal, upstream]
---

The [Drupal CMS](https://new.drupal.org/blog/drupal-cms-2) upstream has been updated to version 2. Newly created sites using the Drupal CMS upstream will now use Drupal CMS 2, which includes a curated set of modules and configuration designed to help you get started faster.

This update only applies to sites created with the Drupal CMS upstream. Sites using the Drupal 10 or Drupal 11 upstreams are not affected. Existing Drupal CMS sites are also not affected — refer to the [Drupal CMS documentation](https://www.drupal.org/docs/drupal-cms) for upgrade guidance.
