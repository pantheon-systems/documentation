published_date: "2026-05-20"
categories: [wordpress, plugins, action-required]
---

Pantheon has released version 2.3.2 of the [WP SAML Auth WordPress plugin](https://wordpress.org/plugins/wp-saml-auth/). This security update fixes vulnerabilities in the `robrichards/xmlseclibs` and `onelogin/php-saml` dependencies. See [#477](https://github.com/pantheon-systems/wp-saml-auth/pull/477) for details.

Update to 2.3.2 from the WordPress dashboard under **Plugins > Installed Plugins**, or download it from the [WordPress Plugin Repository](https://wordpress.org/plugins/wp-saml-auth/).
