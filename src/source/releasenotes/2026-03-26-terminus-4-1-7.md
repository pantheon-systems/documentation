Terminus [4.1.7](https://github.com/pantheon-systems/terminus/releases/tag/4.1.7) is now available. This hotfix release fixes the `domain:verify` command to properly display DNS challenge details.

## Key improvements in this release

- **domain:verify fix**: Fixed the command to display DNS challenge details when a domain is unverified, and added polling for verification status after triggering verification ([#2797](https://github.com/pantheon-systems/terminus/pull/2797))

## How to upgrade to Terminus 4.1.7

If you use Homebrew (macOS-only) to manage your Terminus installation, you should upgrade using:

```shell{promptUser: user}
brew upgrade terminus
```

If you installed Terminus directly from the `.phar` file, you should upgrade using the `self:update` command:

```shell{promptUser: user}
terminus self:update
```

For more information about this release, visit the [GitHub release page](https://github.com/pantheon-systems/terminus/releases/tag/4.1.7).

If you have questions or concerns around Terminus, please use the [Terminus issue queue](https://github.com/pantheon-systems/terminus).
