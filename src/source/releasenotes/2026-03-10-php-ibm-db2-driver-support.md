title: "IBM i Access ODBC driver now available in PHP Runtime Generation 2"
published_date: "2026-03-10"
categories: [infrastructure]
---
The IBM i Access ODBC driver is now available in PHP Runtime Generation 2.

To connect, use [`odbc_connect()`](https://www.php.net/manual/en/function.odbc-connect.php)  
For more information on available PHP extensions and runtime configuration, refer to the [PHP Runtime Generation 2](/php-runtime-generation-2) documentation.
