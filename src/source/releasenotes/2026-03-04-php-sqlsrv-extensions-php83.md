---
title: "Native SQL Server extensions now available for PHP 8.3+"
published_date: "2026-03-04"
categories: [infrastructure]
---
The `sqlsrv` and `pdo_sqlsrv` PECL extensions (v5.13.0) are now available for PHP 8.3 and 8.4. This gives sites native SQL Server connectivity beyond the existing ODBC driver support.

For more information on available PHP extensions and runtime configuration, refer to the [PHP Runtime Generation 2](/php-runtime-generation-2) documentation.
