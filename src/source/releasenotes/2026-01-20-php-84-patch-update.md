PHP version [8.4.17](https://www.php.net/ChangeLog-8.php#8.4.17) is now available on the platform. This security update includes important bug fixes and patches across multiple extensions including Core, DOM, GD, OpenSSL, and Phar. The latest version has already rolled out to all eligible sites.

PHP 8.4 is only available with the new [PHP Runtime Generation 2](/php-runtime-generation-2).
