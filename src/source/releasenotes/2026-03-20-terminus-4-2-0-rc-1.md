Terminus [4.2.0-rc.1](https://github.com/pantheon-systems/terminus/releases/tag/4.2.0-rc.1) is now available as a pre-release. This release candidate merges several standalone plugins into Terminus core and adds Elasticsearch support with unified search commands.

## Key improvements in this release

- **Secrets Manager plugin merged into core**: The [terminus-secrets-manager-plugin](https://github.com/pantheon-systems/terminus-secrets-manager-plugin) is now built into Terminus ([#2764](https://github.com/pantheon-systems/terminus/pull/2764))
- **Repository plugin merged into core**: The [terminus-repository-plugin](https://github.com/pantheon-systems/terminus-repository-plugin) is now built into Terminus ([#2792](https://github.com/pantheon-systems/terminus/pull/2792))
- **Node Logs plugin merged into core**: The [terminus-node-logs-plugin](https://github.com/pantheon-systems/terminus-node-logs-plugin) is now built into Terminus ([#2791](https://github.com/pantheon-systems/terminus/pull/2791))
- **Elasticsearch support**: New `search:enable` and `search:disable` commands for managing search indexing ([#2783](https://github.com/pantheon-systems/terminus/pull/2783))
- **Bug fix**: Empty body should be object, not array ([#2780](https://github.com/pantheon-systems/terminus/pull/2780))

<Alert title="Note" type="info">

The **Secrets Manager**, **Repository** and **Node Logs** plugins will be _deprecated_ after the final release of Terminus 4.2.0 as these features will have been merged into Terminus core. Older versions of Terminus will still require these plugins if the functionality they add is required.

</Alert>

## How to install Terminus 4.2.0-rc.1

This is a pre-release version intended for testing. To install it directly from the `.phar` file, you can download it from the [GitHub release page](https://github.com/pantheon-systems/terminus/releases/tag/4.2.0-rc.1).

For more information about this release, visit the [GitHub release page](https://github.com/pantheon-systems/terminus/releases/tag/4.2.0-rc.1).

If you have questions or concerns around Terminus, please use the [Terminus issue queue](https://github.com/pantheon-systems/terminus).
